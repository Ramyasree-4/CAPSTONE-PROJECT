# Demo Script

1. Log in as an admin and show the dashboard overview.
2. Create a user and assign the `User` role.
3. Upload a PDF or TXT document with department and tags.
4. Show the document moving through processing states.
5. Ask a question in the chat workspace.
6. Open the citation panel and retrieved chunks viewer.
7. Submit positive feedback, then submit a retrieval issue report.
8. Run a RAG evaluation for the answer.
9. Open analytics and show usage, quality, cost, and latency charts.
10. Open audit logs and security logs.

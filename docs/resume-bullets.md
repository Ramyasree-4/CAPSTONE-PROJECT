# Resume Bullet Points

- Built an enterprise RAG SaaS platform using FastAPI, Next.js, MongoDB Atlas, ChromaDB, and multi-provider LLM integrations.
- Designed JWT authentication, refresh tokens, RBAC permissions, audit logging, secure document upload, and prompt-injection detection.
- Implemented a modular ingestion pipeline for PDF, DOCX, TXT, CSV, and PPTX documents with configurable chunking and vector indexing.
- Developed citation-based chat workflows with retrieval diagnostics, conversation history, feedback capture, and RAG evaluation metrics.
- Delivered analytics dashboards for usage, model cost, latency, retrieval quality, hallucination risk, and user activity.

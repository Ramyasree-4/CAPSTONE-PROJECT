# Screenshot Checklist

## Required Screenshots

- [ ] Login page
- [ ] Registration page
- [ ] Dashboard or home page after login
- [ ] Document upload page
- [ ] Document listing or management page
- [ ] Chat / conversational search page
- [ ] Answer with citations displayed
- [ ] Feedback submission or evaluation page
- [ ] Analytics dashboard page
- [ ] Audit logs or security logs page
- [ ] Deployment status or environment configuration page

## Notes

- Capture the UI state with real data where possible.
- Include visible timestamps or labels to show feature behavior.
- If using staging or deployed environment, include the public URL.
